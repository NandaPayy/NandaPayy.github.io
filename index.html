Projects

Meteorite Landings Analysis

This project investigates patterns in meteorite landings using a dataset from NASA's Open Data Portal, focusing on meteorite density across continents. The analysis includes computing probabilities of meteorite sightings, conducting hypothesis tests to determine if meteorite landings are randomly distributed, and comparing distributions across continents using Total Variation Distance (TVD). Additionally, confidence intervals for mean meteorite mass are calculated using the Central Limit Theorem (CLT). Finally, the project explores trends in the median meteorite mass over time to assess any significant changes.

Taylor Swift's Songs Analysis

This project utilizes a dataset of Taylor Swift's songs to build three interactive tools. The first is a song recommender, which takes a user's favorite song and suggests similar tracks from Taylor Swift's discography. The second tool is a lyric searcher that allows users to input a phrase and returns the total number of times it appears across her songs, along with how many songs feature the phrase. The final tool generates a word cloud, summarizing the key themes of each song based on frequently used keywords.

Image Processing App

This project involves the development of a basic image processing app that offers several key features. It includes image processing methods such as negative, grayscale, and rotation. Additionally, the app implements a K-Nearest Neighbors (KNN) classifier for predicting image labels based on processed inputs, providing both visual manipulation and machine learning functionality.
